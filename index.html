การเชื่อมต่อฐานข้อมูลจริง (Google Sheets ผ่าน Apps Script)

ข้อมูลเกณฑ์ประเมินและผลคะแนนทั้งหมดของระบบนี้เก็บอยู่ใน Google Sheets จริง ผ่าน Google Apps Script Web App — ไม่มีการเก็บข้อมูลจำลองไว้ในเบราว์เซอร์อีกต่อไป ระบบ AI ตรวจให้คะแนนจะเรียกใช้ Gemini API จริงผ่านเซิร์ฟเวอร์ Apps Script (คีย์ API เก็บไว้อย่างปลอดภัยฝั่งเซิร์ฟเวอร์ ไม่หลุดมาที่เบราว์เซอร์)

คัดลอก Web App URL ที่ได้จากการ Deploy Apps Script (Code.gs) มาวางที่นี่ — ดูขั้นตอนเต็มในไฟล์ SETUP.md ที่แนบมา

Gemini API Key (สำหรับระบบตรวจ AI)

Key จะถูกส่งไปเก็บที่ฝั่ง Apps Script (Script Properties) — ไม่ถูกเก็บในเบราว์เซอร์ ขอ key ฟรีได้ที่ aistudio.google.com/apikey

เริ่มต้นด้วย AIza — ถ้าขึ้น error 403 ให้ตรวจสอบว่า Gemini API ถูก enable ในโปรเจกต์แล้ว

เปลี่ยนโมเดลได้โดยแก้ค่า GEMINI_MODEL ที่บรรทัดบนสุดของ Code.gs แล้ว Deploy ใหม่ — gemini-2.0-flash ถูกปิดให้บริการแล้ว (1 มิ.ย. 2026)

ส่งชิ้นงานคอมพิวเตอร์/โค้ดพัฒนาโครงสร้างใหม่

กรอกข้อมูลระบบ และเลือกหมวดหมู่เกณฑ์ประเมินที่ตรงกับประเภทผลงาน เพื่อส่งให้เอ็นจิ้น AI ดำเนินการวิเคราะห์โครงสร้างข้อมูล

ไฟล์ข้อความ/โค้ดจะถูกอ่านเนื้อหาแล้วต่อกับช่องด้านบน ส่วนรูปภาพและ PDF จะถูกส่งให้ AI วิเคราะห์ภาพโดยตรง — ไฟล์ทั้งหมดจะถูกเก็บสำเนาไว้ใน Google Drive
